> Type `/Decision` to add a native Confluence decision entry, or maintain the table below. The rule: **no decision lives only in a personal chat** — it is recorded here within the same day.

This log is the durable, searchable record of decisions affecting standards, scope, ownership, or governance. Each entry should let a reader understand *what was decided, why, and who owns it* without reference to chat history.

## How to log a decision

1. Add a row (or a `/Decision` entry) with the next ID.
2. Capture the decision, the options considered, and the rationale.
3. Set the owner and status.
4. Link the related work item / Confluence page.
5. If the decision originated in a chat, copy it here the same day and post the link back in the chat.
